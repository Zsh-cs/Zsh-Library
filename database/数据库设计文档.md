# 数据库设计文档

数据库：zsh_library

单表：book

| 字段名       | 数据类型      | 说明     | 备注 |
| ------------ | ------------- | -------- | ---- |
| id           | bigint        | 主键     | 自增 |
| title        | varchar(255)  | 书名     |      |
| author       | varchar(100)  | 作者     |      |
| isbn         | varchar(20)   | ISBN编号 | 唯一 |
| description  | varchar(255)  | 简介     |      |
| publisher    | varchar(100)  | 出版社   |      |
| publish_date | date          | 出版日期 |      |
| price        | decimal(10,2) | 定价     |      |
